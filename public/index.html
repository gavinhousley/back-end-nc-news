NC News API

This API is for managing content as in posts with comments for a multitude of users.

Available Endpoints:

  • GET /api/topics

    Retrieve a list of all topics

    Example response:

                    
                    {
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

  • GET /api/articles

    Retrieve a list of all articles

    Example response:

    
            {
	"article": {
		"article_id": 1,
		"title": "Running a Node App",

		"votes": 0,
